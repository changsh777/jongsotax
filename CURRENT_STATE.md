## 홈택스 로그인 핵심 패턴 (삽질 방지)

### ✅ 작동하는 방식
```python
# 1. 로그아웃
logout_hometax(page)  # HTML 모달 → [id*=_btn_confirm].click()

# 2. 로그인 (신규/아이디 방식)
login_hometax_id(page, hometax_id, hometax_pw, jumin_raw="XXXXXX-XXXXXXX")
# - 아이디 입력란: #mf_txppWframe_loginboxFrame_iptUserId
# - 2차인증 팝업: [id*=UTXPPABC12][id*=iptUserJuminNo1/2]
# - 확인버튼: [id*=UTXPPABC12][id*=trigger46]

# 3. 신고도움서비스 이동 (goto 금지! SPA)
page.evaluate("document.getElementById('menuAtag_4103080000').onclick()")
time.sleep(4)

# 4. 미리보기 버튼
INDIVIDUAL_PREVIEW_BTN_ID = "#mf_txppWframe_trigger1"
```

### ❌ 하면 안 되는 것
- `page.goto(tmIdx URL)` → "로그인 정보가 없습니다" 에러
- caller에서 `sys.stdout = io.TextIOWrapper(...)` → 신규고객처리.py import 시 충돌
- 로그아웃 직후 바로 goto → 리다이렉트 충돌 (login_hometax_id 내부에서 try/except 처리됨)
