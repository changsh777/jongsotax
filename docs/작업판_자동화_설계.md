# 종소세 작업판 자동화 설계 (2026)

## 파일 구조

| 파일 | 역할 |
|------|------|
| `print_sheet.py` | 고객별 A4 1장 작업준비 레이아웃 생성 (①②③④) |
| `jakupan_gen.py` | 작업판 템플릿에 고객 데이터 자동입력 + 작업준비시트 추가 |

## 템플릿 시트 구성

### 빈양식_간편장부.xlsx
| 시트명 | 사용 케이스 |
|--------|-----------|
| `프리` | 프리랜서 1개만 |
| `프리+프리` | 프리랜서 2개 이상 (업종코드 다른 경우) |
| `사업자+프리` | 사업자등록 1개 + 프리랜서 |
| `사업자+사업자` | 사업자등록 2개 이상 |

### 빈양식_복식부기.xlsx
| 시트명 | 사용 케이스 |
|--------|-----------|
| `프리복식` | 복식부기 + 프리랜서 |
| `사업자복식` | 복식부기 + 사업자등록 |

## 시트 자동 선택 로직

```python
def select_template_and_sheet(biz_rows, 기장의무):
    reg_count  = sum(1 for b in biz_rows if b.get('사업자번호'))  # 등록 사업자 수
    free_count = sum(1 for b in biz_rows if not b.get('사업자번호'))  # 미등록 프리 수
    is_복식 = "복식" in 기장의무

    if is_복식:
        template = TEMPLATE_복식
        sheet = "사업자복식" if reg_count > 0 else "프리복식"
    else:  # 간편장부
        template = TEMPLATE_간편
        if reg_count >= 2:
            sheet = "사업자+사업자"
        elif reg_count == 1 and free_count > 0:
            sheet = "사업자+프리"
        elif free_count >= 2:
            sheet = "프리+프리"
        else:
            sheet = "프리"  # 기본값

    return template, sheet
```

### 판단 근거 (안내문 파싱)
- `기장의무` → "간편장부대상자" / "복식부기의무자"
- `biz_rows` = `parse_anneam_biz()` 결과
  - `사업자번호` 있음 → 등록 사업자
  - `사업자번호` 없음 → 미등록 프리랜서

## 자동입력 가능 필드 (프리 시트 기준)

| 셀 | 내용 | 자동입력 소스 |
|----|------|-------------|
| B2 | 성명 | 고객명 |
| D3 | 복식부기의무자 해당 | 기장의무 파싱 |
| D5 | 간장복식 (6천만이상) | 기장의무 + 수입금액 |
| D6 | 간편장부대상자 해당 | 기장의무 파싱 |
| D9 | 단순율(G등) 해당 | 추계시적용경비율 파싱 |
| D10 | 사업자등록 해당 | 사업자번호 있으면 해당 |
| D11 | 프리랜서 해당 | 미등록 사업소득 or 간이용역소득 |
| E11 | 프리랜서 수입금액 | 안내문 사업장별수입금액 |
| F11 | 업종코드 | 안내문 사업장별수입금액 |
| D12 | 근로 해당/N/A | 안내문 타소득 O/X |
| D13 | 연금 해당 | 안내문 타소득 O/X |
| D14 | 기타 해당/N/A | 안내문 타소득 O/X |
| D15 | 금융 해당 | 안내문 이자/배당 O/X |
| E14/E16 | 기타/주택임대 금액 | 기본값 0 |

## 사람이 채워야 하는 필드

- C19/C20, F19/F20: 신용카드/기타비용 경비
- C29:C44, F29:F44: 경비내역 (당기매입, 급여 등)
- C48:C58: 소득공제 (부양가족 등)
- C60:C61: 세액공제

## 작업준비 시트 (①②③④ 레이아웃)

자동 파싱 → 채워지는 내용:
- ① 종소세 안내문: 수입금액합계, 기장의무, 경비율, 타소득 O/X + 사업장별 수입금액 (사업자번호·업종코드)
- ② 지급명세서/간이용역소득 대조: PDF 파싱 + xlsx 파싱, 차이 계산
- ③ 전년도 소득세: 11개 컬럼 전체 + % 행
- ④ 부가세: 자료있으면 1기/2기, 없으면 "수동조회 필요" 안내
